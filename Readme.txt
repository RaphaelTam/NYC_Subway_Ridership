Does ridership on the NYC subway increase when it rains?  

This project uses statistical tests, linear regression and visualization to answer this question.

Project report, “NYC_subway_ridership_weather.ipynb” is a python notebook file.  

To view the file, copy the NYC_subway_ridership_weather.ipynb link on github and paste it to the input box at http://nbviewer.ipython.org. Hit Go!
